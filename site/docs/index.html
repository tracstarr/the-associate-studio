Documentation

Everything you need to know about The Associate Studio.

The Associate Studio is a native Windows desktop IDE built around Claude Code CLI. It provides a multi-session terminal environment, real-time project tracking, and integrated access to GitHub, Linear, and Jira — all in a single window.

Built with Tauri v2 (Rust backend) and React 19 (TypeScript frontend). Approximately 10MB as a standalone binary.

Technical Reference

The source repository's docs/ directory contains detailed technical documentation maintained alongside the code. These markdown files are the primary reference for contributors and cover implementation decisions, format specifications, and build system details.

  • architecture.md Component tree, data flow, state management, watcher system
  • tech-stack.md Every library and crate with version, purpose, and decision rationale
  • build.md Build toolchain, PATH requirements, MSYS2/MinGW setup, crate-type gotchas
  • terminal.md PTY implementation, portable-pty, ConPTY, nested Claude detection
  • data-formats.md Structure of ~/.claude/ directories, JSONL formats, session index
  • integrations.md Auth flows, API endpoints, credential storage implementation
  • security.md Windows Credential Manager usage, DPAPI, secret handling policy
  • theming.md Color palette, CSS custom properties, design system
  • keybindings.md Shortcut implementation, input guard logic, terminal-safe bindings

These files are kept in sync with the codebase. When a significant implementation decision is made — a new library chosen, a format quirk discovered, a security boundary drawn — the relevant doc is updated as part of the same change.
